Model File

Upload a new .pt model file (e.g. best.pt, latest_best.pt). Replaces existing file with same name or adds alongside other weights. Re-encrypted automatically.

Regex ON/OFF

When checked: OCR output passes through plate format regex, state code correction and digit/character substitution (e.g. O→0, I→1).
When unchecked: raw OCR text used as-is after basic alphanumeric stripping — useful when plates don't follow standard Indian format.

Frame Processing

Process every Nth frame from the video stream.
1 = every frame (max accuracy, high CPU). 3 = every 3rd frame (default, balanced). Higher = less CPU but slower plate detection.

Detection Bounding Box

Increase if digits are cut on tilted plates. Added on left and right.
Increase if digits are cut on top or bottom. Added on top and bottom.

Confidence Thresholds

Detections below this score are ignored during fast 640-pass inference. Higher = fewer false positives, lower = more sensitive.
Fallback 1280-pass threshold (runs only when 640-pass OCR is incomplete). Can be slightly lower to catch difficult plates.
